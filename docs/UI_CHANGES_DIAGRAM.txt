5. REMOVED ACTIONS FROM RECORDING
══════════════════════════════════

These actions are NO LONGER recorded in macros:
✗ delete           (Usuń)
✗ duplicate        (Duplikuj)
✗ copy             (Kopiuj)
✗ cut              (Wytnij)
✗ paste_before     (Wklej przed)
✗ paste_after      (Wklej po)
✗ swap             (Zamień miejscami)
✗ insert_blank_*   (Wstaw pustą stronę)

These actions ARE STILL recorded:
✓ rotate_left/right        (Obróć)
✓ select_all/odd/even/...  (Zaznacz)
✓ shift_page_content       (Przesuń zawartość)
✓ insert_page_numbers      (Dodaj numerację)
✓ remove_page_numbers      (Usuń numerację)
✓ apply_page_crop_resize   (Kadruj/Zmień rozmiar)


6. IMPLEMENTED REPLAY FUNCTIONS
════════════════════════════════

✓ _replay_insert_page_numbers()
  - Full implementation with all parameters
  - Supports all 4 rotation angles
  - Handles all alignment modes (left/right/center)
  - Mirror margins support

✓ _replay_remove_page_numbers()
  - Pattern-based detection
  - Top and bottom margin support
  - Regex matching for various formats

✓ _replay_apply_page_crop_resize()
  - All crop modes supported
  - All resize modes supported
  - Position and offset handling
